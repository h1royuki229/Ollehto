## How to Start
1. Compile ---> `sh compile_src.sh`
2. Change current directory ---> `cd src`
3. Run Server ---> `java ServerProgram`
4. Run Client ---> `java ClientProgram {Server's IP address}`
5. Enter Your Name at StartPage
6. You will get MainPage
